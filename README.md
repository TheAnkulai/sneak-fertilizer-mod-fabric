<p>This mod allows you to fertilize saplings and crops using the “shift” key.
The effect has no time limit between presses. </p><br>
<p>Fertilizer radius of action - 5 blocks.</p> <br>
<p>This mod is inspired by an old mod that was in the stoneblock modpack. I couldn't find the same one for newer versions, so I decided to make one myself.</p>
